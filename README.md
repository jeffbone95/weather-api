##About 
``` 
This weather APi is designed for the user to enter a city destination and get a weather forecast 

## User Story 
As a user, I want to be able to enter a city of my choosing and see the weather there. 

## Acceptance Criteria 
Given I am searching for the weather 
When I enter the city of my choice 
Then I am presented with current weather conditions  
When I need specifics 
Than I am presented with temperature, wind speed, and humidity
